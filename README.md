Rock Paper Scissors

A simple Rock Paper Scissors game I built to practice JavaScript logic and DOM manipulation. Nothing fancy — just a solid way to train my coding brain.
Play it here

👉 Live on Netlify
https://rock-paper-scissors-202.netlify.app/

How to play

    Click on rock, paper, or scissors

    The computer makes its choice

    Winner gets a point

    First to 5 wins (or however you set it up)

What I learned

    How to compare game choices without messing up the logic

    Managing game state (scores, rounds, game over)

    Updating the UI without refreshing the page

    Debugging those weird edge cases (like when both pick the same)

Run locally

Just clone and open index.html in your browser — no dependencies, no npm, no headache.
bash

git clone https://github.com/yourusername/rock-paper-scissors.git
cd rock-paper-scissors
